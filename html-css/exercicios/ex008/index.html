Reproduzindo áudio

Preload

Indica se o audio será ou não pré-carregado e aceita 3 valores:

  • metadata - Carrega apenas as informações sobre o arquivo(tamanho, tempo, informações de direitos e etc)
  • none - Não carregará até o usuário dar play.
  • auto - Padrão, vai carregar o arquivo de audio inteiro assim que o usuário entrar no site, mesmo que não dê play.
